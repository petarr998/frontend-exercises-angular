# Collapsible Tree

## Target of task

Implement design for the tree elements.

## Key points

- Implement as close as possible to the design (image base-d3-js-design in assets -> design folder).
- use scss best practice for dynamic styles.
- Pay attention for clear and short commits, comments and code structure
- Each group get different color (not important which color)
- All children of a group should have the color of the parent group

## First step

Fix width of tree to catch the full width of screen with horizontal scroll if needed
